Phantom

Phantom is a free point-and-click adventure for the web based loosely off the Phantom Menace and done in the style of the 8-bit NES game Shadowgate.

Play Phanom now!

History

Phantom was originally intended for An Event Apart's 10k contest. However, the amount of graphics and text we wanted made it difficult to get the game below ten kilobytes without sacrificing too much content. We were able to get the game below 14k but ran out of time to squeeze it down more and were not able to submit it.

Style

Old NES games are awesome and so are old PC point-&-click adventures. It should go with out saying then that Shadowgate, a point-&-click adventure on the NES, is doubly awesome. After seconds of thought we realized that a first-person adventure game is perfect for the web, a place where you point as well as click. Thus began the inspiration for our 10k entry. Our goal was an 8-bit, first-person, point-&-click adventure based on Shadowgate's style.

All graphics are original pixel art by Moose at Play

Script

A game engine alone does not make a good game. We needed content but were too busy staring at pixels and code to think up anything interesting. The project lead remembered a great website, The Editing Room, that writes silly, abridged scripts for popular movies and thought their Phantom Menace script would work nicely. We contacted the author and he said, "Go for it," so we did.

Not intending to complete graphics and scenes for the entire abridged script, the Internet now has: Star Wars: Episode I – The Phantom Menace: The Abridged Script: The Abridged Game. We hope you like it!
